# College Alumni Network Portal

Full Stack Web Development internship project using MERN stack.

## Features
- User registration/login
- Alumni directory
- Jobs module
- Events module
- Community posts
- Profile update

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Run
### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```
